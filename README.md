# Airbnb Data Engineering Assessment

Data engineering analysis of Inside Airbnb data for London, United Kingdom.

## Project Structure

- `notebooks/` — Jupyter notebooks (numbered by execution order)
- `src/` — Reusable Python modules
- `sql/` — SQL schema and analytical queries
- `data/` — Raw and processed datasets
- `reports/` — Final report and figures
- `docs/` — AI usage disclosure, decisions log, prompts log

## Setup

1. Clone this repository
2. Create virtual environment: `python -m venv venv`
3. Activate it: `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Open VS Code and run notebooks in order: `01_*` → `08_*`
