## Project Overview

This project demonstrates a complete data engineering pipeline applied to the Inside Airbnb dataset for London. It covers data ingestion, profiling, cleaning, dimensional modeling (star schema), exploratory data analysis with 15 visualizations, and statistical hypothesis testing with effect size reporting.

**Key Results:**

- 96,182 listings and 1,887,519 reviews analyzed
- 4-table star schema (3 dimensions + 1 fact table)
- 4 hypothesis tests, all statistically significant
- 15 visualizations with business interpretations
- Counterintuitive finding: high-review listings have lower prices

## Setup Instructions

### Prerequisites

- Python 3.11 or higher
- Git
- VS Code (recommended) or any Jupyter-compatible IDE

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yohansmith/airbnb-data-engineering-assessment.git
cd airbnb-data-engineering-assessment

# 2. Create virtual environment
python -m venv venv

# 3. Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 4. Install dependencies
pip install -r requirements.txt
```

## Execution Guide

**Run notebooks in numerical order (01 → 06).** Each notebook builds on outputs from previous ones.

### Quick Start (5 minutes)

```bash
# Open in VS Code
code .

# Run notebooks in order: 01 → 06
# Each notebook should run end-to-end without errors
```

### Detailed Execution

1. **Notebook 01:** Downloads raw data from Inside Airbnb public source, loads into DuckDB
2. **Notebook 02:** Profiles data quality, generates quality report
3. **Notebook 03:** Cleans and standardizes data, creates `listings_final` table
4. **Notebook 04:** Implements star schema with dimension and fact tables
5. **Notebook 05:** Generates 15 EDA visualizations
6. **Notebook 06:** Runs statistical hypothesis tests with effect sizes

**Expected runtime:** 10-15 minutes total on a modern laptop.

### Output Files Generated

After running all notebooks, you will have:

- `data/airbnb.db` — DuckDB database with all tables
- `reports/figures/*.png` — 15 visualization files
- Updated `docs/data_quality_report.md` with findings

## Privacy and Data Handling

This project follows privacy best practices:

- **No credentials or API keys** are included in this repository
- **No personal data** beyond the publicly available Inside Airbnb dataset
- **No sensitive information** (passwords, tokens, internal URLs)
- **Dataset is publicly sourced** from insideairbnb.com
- **Host names are pseudonyms** in the source data (Airbnb's default behavior)
- **All data files are gitignored** — only code is version controlled

If you need to test with sensitive data, use environment variables and add them to `.gitignore` before committing.
